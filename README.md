# MAATKFM - 天下布魔自動化腳本
這是一個基於 MAAFramework 開發的《天下布魔》自動化腳本專案。旨在透過圖形識別與 ADB 指令，達成智慧化的遊戲日常任務管理，減少重複性操作。

環境需求

MAAFramework: v5.1.0 或以上版本。

模擬器: 建議使用 MuMu Player 12。

解析度: 建議設置為 1920x1080 確保圖像識別準確率。

📦 安裝與使用

下載專案：

前往 Releases 頁面下載最新的打包版本 (MAATKFM.zip)。

解壓縮後開啟 MFAAvalonia.exe
